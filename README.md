# Capstone Lightning Talk 1 — Project Proposals

GA Data Science Bootcamp (Bahrain) — Capstone Part 1: Pitch + Problem Statement

## Overview

This repository contains the EDA notebook and slide deck for three potential capstone project topics, each including a problem statement, goal, data dictionary, and exploratory data analysis.

## Topics

1. **Air Quality Prediction** — Regression model predicting PM2.5 concentration using the Beijing Multi-Site Air Quality dataset (420,768 rows × 18 columns)
2. **Solar Power Generation Forecasting** — Regression model predicting solar energy output (Wh) from weather and irradiance data (196,776 rows × 17 columns)
3. **Motor Fault Detection (MCSA)** — Classification model identifying motor condition (Healthy / BRB / SWF) from raw three-phase current signals (2.7M rows × 18 columns), based on Bruinsma et al. (2024) and extending prior senior thesis research

## Contents

- `lightning-talk-notebook.ipynb` — Problem statements, data dictionaries, and EDA for all three topics
- `capstone_lightning_talk.pptx` — Slide deck summarizing each topic's problem statement, goal, and EDA findings
